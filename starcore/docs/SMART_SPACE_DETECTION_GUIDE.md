## 🧠 什么是智能空间检测？

**智能空间检测**是一种革命性的村民AI优化技术，系统会自动检测村民周围的可活动空间大小，并根据空间大小智能调整优化程度。

### 核心理念

> **空间越小，AI越少**

- 密闭空间（1x1商店）→ 完全移除AI
- 小空间（3x3房间）→ 高度优化
- 中等空间（10x10建筑）→ 中度优化
- 大空间（村庄）→ 轻度优化
- 开放空间（野外）→ 不优化

## 🔍 工作原理

### 1. 空间检测算法

```java
// 伪代码
function calculateSpace(villager):
    airBlocks = 0
    radius = 8
    
    // 检测8x5x8范围（横向8，纵向5）
    for x in [-radius, radius]:
        for y in [-2, 3]:
            for z in [-radius, radius]:
                if isPassable(x, y, z):
                    airBlocks++
    
    return airBlocks
```

**检测范围：**
- 横向：±8方块
- 纵向：-2到+3方块
- 总共：17x5x17 = 1445个方块位置

---

### 2. 优化决策流程

```
村民生成/区块加载
    ↓
异步检测周围空间
    ↓
计算可活动方块数
    ↓
0-50方块 → MAXIMUM (完全移除AI)
51-150   → HIGH     (高度优化)
151-500  → MEDIUM   (中度优化)
501-1000 → LOW      (轻度优化)
1000+    → NONE     (不优化)
    ↓
应用对应优化
    ↓
记录统计信息
```

---

### 3. 实时调整

**重新检测触发条件：**
- 村民传送
- 村民移动超过10格
- 定期检测（每5分钟）

**自动调整：**
```
村民从开放空间 → 进入小房间
优化等级: NONE → HIGH
AI自动移除 ✓
```

**缺点：**

- ⚠️ 不检测活板门状态

**总体影响：**
