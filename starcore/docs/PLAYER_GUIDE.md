# STARCORE 玩家新手指南

欢迎来到 STARCORE！这是一个国家战略模拟插件，你可以创建国家、圈地保护、发展科技、外交结盟、甚至发动战争！

**阅读时间：10 分钟** | **上手时间：5 分钟**

---

## 🚀 10 分钟快速上手

### 第一步：创建你的国家（2分钟）

**准备工作：**
- 确保你有足够的金币（默认需要 10,000 金币）
- 找一个好位置作为你的首都（建议在你的基地）

**开始创建：**
```
/sc n c 我的国家名称
```
或
```
/starcore nation create 我的国家名称
```

**成功标志：**
- 你脚下会出现一个信标（国家核心）
- 收到创建成功的消息
- 扣除创建费用

💡 **提示：** 国家名称不能与其他国家重复，支持中文和英文。

---

### 第二步：圈地保护你的领土（3分钟）

Minecraft 的地图被分为 16×16 的区块（Chunk），按 `F3+G` 可以看到区块边界。

**方式一：快速圈地（推荐新手）**
```
/sc n cl
```
站在你想保护的区块，输入这个命令，当前区块就被圈地了。

**方式二：批量圈地（推荐熟练玩家）**
```
/sc n t          # 获取圈地工具
```
你会得到一个木棍（圈地工具）：
1. **左键点击**第一个角落的方块
2. **右键点击**对角线的方块
3. 输入 `/sc n ok` 确认圈地
4. 如果想取消，输入 `/sc n x`

**查看圈地结果：**
```
/sc n ls         # 查看你的国家领地列表
/sc n here       # 查看当前区块的主人
```

**费用说明：**
- 圈地需要花费金币（默认 500 金币/区块）
- 距离世界中心越远，费用越高
- 不同生物群系价格不同（沙漠便宜，森林贵）

💡 **提示：** 被圈的地只有你的国家成员能建造，其他人无法破坏！

---

### 第三步：邀请朋友加入（1分钟）

**让朋友加入你的国家：**
```
# 朋友输入：
/sc n join 你的国家名称
```

然后你的国家会收到一个"决议"（投票），通过后朋友就加入了。

**查看国家成员：**
```
/sc n i          # 查看你的国家信息
```

**管理成员：**
```
/sc n kick 玩家名  # 踢出成员（需要投票）
```

---

### 第四步：发展你的国家（3分钟）

#### 💰 管理国库
```
/sc tr s         # 查看国库状态
/sc tr d 1000    # 存入 1000 金币到国库
/sc tr w 500     # 从国库取出 500 金币
```

#### 🏭 激活国策
国策可以给你的国家提供各种加成（如金币收入+20%，战斗力+15%等）。

```
/sc po t         # 打开国策树 GUI
```
或
```
/starcore 国策 菜单
```

在 GUI 中点击你想激活的国策，确认消耗即可。

**常见国策：**
- **重商主义** - 金库收入 +20%
- **精兵政策** - 战斗力 +15%
- **开放边境** - 外交关系改善速度 +30%

#### 🔬 研发科技
```
/sc tech t       # 打开科技树 GUI
```

科技需要消耗国库金币和资源，但可以解锁强大的能力。

#### 🤝 外交关系
```
/sc dip s        # 查看外交状态
/sc dip set 国家名 友好  # 设置外交关系
```

外交关系等级：
- **中立** - 默认关系
- **友好** - 可以在对方领地建造
- **同盟** - 可以共同作战
- **敌对** - 无法交易
- **战争** - 可以攻击对方
- **附庸** - 从属关系

---

### 第五步：扩张和发展（1分钟）

#### 🗺️ 查看地图
```
/sc m w          # 获取个人地图链接
```
或
```
/starcore 地图 网页
```

你会收到一个专属链接，在浏览器打开可以看到：
- 你的国家领地
- 其他国家位置
- 资源分布
- 还能在地图上直接圈地！

#### 🏛️ 创建城邦
城邦是国家的子区域，可以有独立管理。
```
/sc n city c 城邦名称
```

#### ⚔️ 发动战争
```
/sc w d 敌国名称  # 宣战（需要国家领袖权限）
/sc w s          # 查看战争状态
/sc w e 敌国名称  # 结束战争
```

## 🎮 进阶玩法

### 资源系统
国家领地内会生成资源区块，定期刷新矿物。

```
/sc rsc s        # 查看资源状态
/sc rsc d        # 查看资源区块列表
/sc rsc i        # 查看当前区块资源详情
/sc rsc m        # 迁移资源区块
```

**资源作用：**
- 用于科技研发
- 用于高级建筑
- 可以交易给其他国家

### 官员系统
国家可以任命官员，分配不同权限。

```
/sc off s        # 查看官员列表
/sc off a <玩家> <职位>  # 任命官员
/sc off rm <玩家>        # 罢免官员
```

**常见职位：**
- **财政大臣** (treasurer) - 管理国库
- **外交官** (diplomat) - 管理外交
- **元帅** (marshal) - 指挥战争
- **内务大臣** (steward) - 管理内政

### 决议系统
重要决策需要投票通过。

```
/sc res ls       # 查看待投票的决议
/sc res s <编号> # 投票支持
```

## ❓ 常见问题 FAQ

### Q: 我创建了国家，但圈不了地？
**A:** 检查以下几点：
1. 你是国家成员吗？用 `/sc n i` 查看
2. 你有足够的金币吗？用 `/sc eco b` 查看余额
3. 这块地是否被其他国家占领？用 `/sc n here` 查看
4. 国家是否达到圈地上限？用 `/sc rsc s` 查看容量

### Q: 如何邀请朋友加入国家？
**A:** 让你的朋友输入 `/sc n join 你的国家名称`，然后国家需要通过决议。

### Q: 国策激活后多久生效？
**A:** 立即生效，你可以在 `/sc po s` 中看到当前激活的国策。

### Q: 如何改变国家名称？
**A:** 使用 `/sc n rename 新名称`，需要支付一定费用。

### Q: 战争如何发起？
**A:** 只有国家领袖或元帅可以宣战，使用 `/sc w d 敌国名称`

### Q: 如何查看我的国家有多少钱？
**A:** 使用 `/sc tr s` 查看国库状态

### Q: 圈地后其他玩家还能进入吗？
**A:** 可以进入，但不能破坏和建造。

### Q: 如何解散国家？
**A:** 使用 `/sc n disband`，注意这会删除所有领地和数据！

### Q: 国家有人数限制吗？
**A:** 有的，默认是 50 人，可以通过升级国家等级增加上限。

### Q: 如何升级国家等级？
**A:** 国家通过积累经验自动升级，经验来源包括：
   - 资源刷新
   - 成员在线时间
   - 完成任务

## 💡 游戏技巧

### 新手技巧
1. **先圈重要区域** - 优先保护你的基地和资源点
2. **善用地图** - Web 地图可以看到全局，规划更方便
3. **加入联盟** - 单打独斗很难，找盟友很重要
4. **平衡发展** - 不要只圈地，记得发展经济和科技
5. **看准时机宣战** - 战争前确保国库充足、科技领先

### 经济技巧
1. **合理存款** - 个人金币不会丢失，但国库可以被抢
2. **税收优化** - 设置合理的税率，过高会导致成员流失
3. **资源交易** - 缺少的资源可以和其他国家交易
4. **国策配合** - 经济国策+商业建筑效果更好

### 外交技巧
1. **先友好后同盟** - 不要一开始就结盟，观察一段时间
2. **保持中立** - 不是所有国家都要表态
3. **外交缓冲** - 敌对前先降级到中立，给对方反应时间
4. **文书往来** - 重要外交决定可以在国家公告中说明

### 战争技巧
1. **情报先行** - 用地图查看敌国布局
2. **经济战** - 切断敌国资源供应
3. **快速决战** - 战争消耗大，速战速决
4. **守城为主** - 防守比进攻容易

- 🗺️ **探险家** - 圈地覆盖多个生物群系
- ⚔️ **战神** - 赢得最多场战争
