# StarCore 国家管理菜单使用指南

## 快速开始

### 打开菜单

玩家可以通过多种方式打开国家管理菜单：

```
/menu              # 主菜单命令
/mainmenu          # 别名
/星核菜单           # 中文别名
```

**前提条件**: 玩家必须已加入一个国家

## 菜单功能

### 主菜单布局

```
┌─────────────────────────────────┐
│    ⚔ [国家名称] 国家管理         │
├─────────────────────────────────┤
│                                 │
│  [灯塔]    [头颅]    [地图]     │
│  国家信息   成员管理   领土管理   │
│                                 │
│           [屏障]                │
│            关闭                 │
└─────────────────────────────────┘
```

### 功能按钮

#### 🏛️ 国家信息
- **图标**: 灯塔 (Beacon)
- **功能**: 查看国家详细信息
- **执行**: `/sc nation info`

#### 👥 成员管理
- **图标**: 玩家头颅
- **功能**: 管理国家成员（开发中）
- **状态**: 🚧 即将推出

#### 🗺️ 领土管理
- **图标**: 地图
- **功能**: 查看和管理领土
- **执行**: `/sc nation claims`

#### ❌ 关闭
- **图标**: 屏障
- **功能**: 关闭菜单

## 权限

所有国家成员都可以查看国家信息，特定管理功能需要相应权限：

- `starcore.nation.member` - 基础成员权限（查看信息）
- `starcore.nation.officer` - 官员权限（成员管理）
- `starcore.nation.owner` - 国主权限（完全管理）

## 常见问题

### Q: 为什么我无法打开菜单？
A: 请确认：
1. 你已经加入了一个国家（使用 `/nation info` 检查）
2. 服务器已安装 StarCore 插件
3. 你有基础权限 `starcore.nation.member`

### Q: 菜单中的按钮点击没反应？
A: 这通常是权限问题。请联系服务器管理员检查权限配置。

### Q: 可以自定义菜单样式吗？
A: 服务器管理员可以在插件配置中修改菜单样式（需要修改源码）。

## 管理员配置

### 配置文件位置
```
plugins/STARCORE/config.yml
```

### 菜单配置项
```yaml
menu:
  # 菜单提供者类型
  # auto - 自动检测最佳提供者
  # triumph - 强制使用箱子界面
  # fallback - 使用聊天菜单
  provider: auto
```

### 重载配置
```
/starcore reload
```
