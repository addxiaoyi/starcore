#### 1. 生物堆叠集成 ⭐新增
**目标：** 减少实体数量，提升性能

**推荐方案：** StackMob（开源经典）
- GitHub: https://github.com/ploppyperson/StackMob-5
- SpigotMC: https://www.spigotmc.org/resources/29999/
- Modrinth: https://modrinth.com/project/EqaioisK

**集成方式：**
```yaml
# plugin.yml
softdepend: [StackMob, MobStacker, RoseStacker]
```

```java
// MobStackIntegration.java
public class MobStackIntegration {
    private boolean stackMobAvailable = false;
    private boolean mobStackerAvailable = false;
    private boolean roseStackerAvailable = false;
    
    public void init() {
        PluginManager pm = plugin.getServer().getPluginManager();
        
        stackMobAvailable = pm.getPlugin("StackMob") != null;
        mobStackerAvailable = pm.getPlugin("MobStacker") != null;
        roseStackerAvailable = pm.getPlugin("RoseStacker") != null;
        
        if (stackMobAvailable) {
            plugin.getLogger().info("✅ StackMob 集成已启用");
        } else if (mobStackerAvailable) {
            plugin.getLogger().info("✅ MobStacker 集成已启用");
        } else if (roseStackerAvailable) {
            plugin.getLogger().info("✅ RoseStacker 集成已启用");
        }
    }
}
```

**工作量：** 0.5天  
**文件：** 2个类
