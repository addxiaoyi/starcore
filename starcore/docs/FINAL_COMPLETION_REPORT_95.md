%starcore_best_killstreak% - 最高连杀

%starcore_damage_dealt% - 造成伤害

## 🚀 使用指南

### 快速开始

1. **编译插件**
   ```bash
   mvn clean package
   ```

2. **安装到服务器**
   ```bash
   cp target/starcore.jar server/plugins/
   ```

3. **安装可选依赖**
   - Vault（经济系统）
   - PlaceholderAPI（数据显示）
   - StackMob（性能优化）

4. **启动服务器**
   ```bash
   java -Xms2G -Xmx4G -jar paper.jar
   ```

5. **测试功能**
   ```
   /menu - 打开主菜单
   /duel <玩家> - 发起决斗
   /pvpstats - 查看统计
   ```
